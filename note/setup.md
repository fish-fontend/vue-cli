## Project setup
```
npm install
```

### Compiles and hot-reloads for development
``` bash
npm run serve
```

### Compiles and minifies for production
``` bash
npm run build
# vue-cli-service build --modern
# 这个参数可以让编译中不再兼容太不支持es2015的浏览器。提高了性能
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
